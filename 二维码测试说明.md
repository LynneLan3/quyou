## 🔍 问题分析

根据截图显示的诊断信息：

```
失败网址：
https://quyou.vercel.app/quiz/a0000000-0000-0000-0000-000000000004?from=iYTFiMmMzZDQtZTVmNi03ODkwLWFiY2QtZWYxMjM0NTY3ODkw

失败原因：
net::ERR_CONNECTION_REFUSED
```

### 问题点

1. **`from` 参数异常**：
   - 实际值：`iYTFiMmMzZDQtZTVmNi03ODkwLWFiY2QtZWYxMjM0NTY3ODkw`
   - 预期值：`f.YTFiMmMzZDQtZTVmNi03ODkwLWFiY2QtZWYxMjM0NTY3ODkw`
   - **缺少了 `f.` 前缀！**

2. **错误类型**：
   - `ERR_CONNECTION_REFUSED` 通常表示网络连接被拒绝
   - 但这里更可能是 URL 格式错误导致浏览器无法正确解析

## 🔧 可能的原因和解决方案

### 原因 1: 二维码生成库的问题

**可能性**：`qrcode.react` 在处理某些字符时可能有问题

**解决方案**：尝试 URL 编码

```tsx
<QRCodeSVG 
  value={encodeURI(shareUrl)}  // 添加 encodeURI
  size={80}
  bgColor="#FFFFFF"
  fgColor="#000000"
  level="M"
  includeMargin={false}
/>
```

### 原因 2: 特殊字符处理

**可能性**：URL 中的 `.` 字符可能被错误处理

**解决方案**：改用其他分隔符

```typescript
// 在 src/lib/shareLink.ts 中
const PREFIX = 'f_';  // 改用下划线而不是点
```

### 原因 3: 二维码尺寸或纠错级别

**可能性**：URL 太长，二维码无法完整编码

**解决方案**：
1. 提高纠错级别到 Q 或 H
2. 增大二维码尺寸到 100px 或更大

```tsx
<QRCodeSVG 
  value={shareUrl}
  size={100}  // 增大尺寸
  level="Q"   // 提高纠错级别
  ...
/>
```

### 原因 4: html-to-image 截图问题

**可能性**：生成分享卡片图片时，二维码被压缩或失真

**解决方案**：提高截图质量

```typescript
const dataUrl = await htmlToImage.toPng(shareCardRef.current, {
  cacheBust: true,
  pixelRatio: 3,  // 提高到 3
  quality: 1.0,   // 最高质量
});
```

## 🎯 立即尝试的解决方案

基于截图中 `from` 参数缺少 `f.` 的情况，我怀疑是二维码生成或扫描时的问题。

### 方案 1: 改用下划线分隔符（推荐）

修改 `src/lib/shareLink.ts`：

```typescript
const PREFIX = 'f_';  // 改用下划线

export function encodeFromParam(userId: string): string {
  if (!userId) return '';
  try {
    const base64 = btoa(unescape(encodeURIComponent(userId)));
    const safe = base64.replace(/\+/g, '-').replace(/\//g, '_').replace(/=+$/, '');
    return PREFIX + safe;
  } catch {
    return userId;
  }
}

export function decodeFromParam(encoded: string | null | undefined): string | null {
  if (encoded == null || encoded === '') return null;
  try {
    let raw = encoded;
    if (raw.startsWith(PREFIX)) raw = raw.slice(PREFIX.length);
    // ... 其余代码不变
  } catch {
    return encoded;
  }
}
```
