# 二维码修复说明

## 🎯 问题描述

**用户反馈**：
- ✅ 复制链接功能可以正常实现匹配功能
- ❌ 分享卡片的二维码无法扫描跳转到对应页面

**需求**：
- 将二维码改为方形带圆角展示
- 保证受邀人扫描成功跳转到填写问卷的页面

## 🔍 问题分析

### 原始设计的问题

**1. 圆形容器裁剪了二维码内容**
```tsx
// ❌ 问题代码
<div className="w-20 h-20 bg-white rounded-full p-3 ...">
  <div className="w-full h-full rounded-full overflow-hidden ...">
    <QRCodeSVG value={shareUrl} size={56} ... />
  </div>
</div>
```

**问题**：
- `rounded-full` 创建完美圆形
- `overflow-hidden` 裁剪超出圆形的内容
- 二维码的四个角被裁剪，导致部分数据丢失
- 扫描器无法识别不完整的二维码

**2. 尺寸过小**
- 容器：80x80 px（20 * 4）
- 内边距：12px（p-3）
- 实际二维码：56px
- 手机扫描时可能太小，识别困难

**3. 纠错级别过低**
- 使用 `level="L"`（低纠错，约 7% 容错）
- 当部分二维码被遮挡或损坏时，无法恢复数据

## ✅ 解决方案

### 修改后的设计

```tsx
// ✅ 修复后的代码
<div className="w-24 h-24 bg-white rounded-xl p-2 ...">
  <QRCodeSVG 
    value={shareUrl} 
    size={80}
    bgColor="#FFFFFF"
    fgColor="#000000"
    level="M"
    includeMargin={false}
  />
</div>
```

### 关键改动

| 属性 | 修改前 | 修改后 | 原因 |
|------|--------|--------|------|
| 容器形状 | `rounded-full` | `rounded-xl` | 方形圆角，不裁剪二维码 |
| 容器尺寸 | `w-20 h-20` (80px) | `w-24 h-24` (96px) | 增大尺寸，易于扫描 |
| 内边距 | `p-3` (12px) | `p-2` (8px) | 减小边距，留更多空间给二维码 |
| 二维码尺寸 | `size={56}` | `size={80}` | 增大 43%，提高识别率 |
| 背景色 | `transparent` | `#FFFFFF` | 白色背景，提高对比度 |
| 纠错级别 | `level="L"` (7%) | `level="M"` (15%) | 中等纠错，提高容错能力 |
| 边框 | `border-4` | `border-2` | 减小边框，留更多空间 |
| 内部容器 | 有 `overflow-hidden` | 移除 | 避免裁剪 |

## 📊 技术细节

### QR Code 纠错级别

| 级别 | 容错率 | 适用场景 |
|------|--------|----------|
| L | ~7% | 理想环境，无遮挡 |
| M | ~15% | 一般使用，轻微遮挡 ✅ 当前使用 |
| Q | ~25% | 可能有遮挡或损坏 |
| H | ~30% | 严重遮挡或需要嵌入 Logo |

**选择 M 级别的原因**：
- 平衡了数据容量和容错能力
- 即使部分二维码被遮挡，仍可识别
- 不会因为纠错数据过多而使二维码过于密集

### 尺寸计算

**修改前**：
```
容器：80px (w-20)
内边距：12px (p-3)
可用空间：80 - 12*2 = 56px
二维码：56px
```

**修改后**：
```
容器：96px (w-24)
内边距：8px (p-2)
可用空间：96 - 8*2 = 80px
二维码：80px
```

**增长率**：80/56 ≈ 1.43（增大 43%）

### 圆角设计

- `rounded-xl` = `border-radius: 0.75rem` (12px)
- 在 96px 的容器上，圆角占比 12.5%
- 既保持美观，又不影响二维码识别

## 📱 移动端扫描最佳实践

### 二维码生成建议
1. **尺寸**：至少 80x80px，推荐 100x100px 以上
2. **对比度**：黑色前景 + 白色背景
3. **纠错级别**：M 或 Q 级别
4. **留白**：周围留出适当空白（quiet zone）
5. **清晰度**：避免模糊或失真

### 扫描环境要求
1. **光线**：充足但不过曝
2. **距离**：10-30cm 之间
3. **角度**：尽量垂直对准
4. **稳定性**：保持手机稳定
